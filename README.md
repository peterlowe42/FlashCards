# FlashCards
Code challenge for Articulate

## Install Steps
- clone repo 

### In command line
 - cd into rails_api dir
 - $ bundle install
 - $ rake db:migrate db:seed
 - $ rails s
### in seperate teminal window
 - cd into front_end dir
 - $ npm install
 - $ npm run start
 
 - View on localhost:5000
